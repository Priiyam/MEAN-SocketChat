# MEAN-SocketChat

Relatime Chat application using Socket.io, Node.js, Angular 4 and MongoDB

### Version
1.0.0

## Install Dependencies
```bash
npm install 
```

## Run Server
```bash
npm start
```

## Run App
Open index.html
